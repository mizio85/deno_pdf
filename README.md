# Deno PDF Generator

This is a simple PDF generator for Deno, inspired by `pdfmake`. It uses `pdf-lib` under the hood.

## Features

- A4/Letter page sizes with portrait/landscape orientation.
- Automatic text wrapping and page overflow.
- Horizontal and vertical alignment for text.
- Flexible table layouts with dynamic column widths and row heights.
- Customizable table styling (fill color, borders), including conditional formatting.
- Page footers with page number placeholders.

## Using as a Module

To use this PDF generator in your own Deno project, you can import the `createPdf` function from the `mod.ts` file.

### Remote Import (from a URL)

If you have hosted this module on a service like GitHub, you can import it directly via its raw URL.

```typescript
// Replace with the actual raw URL to your mod.ts file
import { createPdf } from 'https://deno.land/x/your_module/mod.ts';
```

### Local Import

If you have the code locally in your project, you can use a relative path.

```typescript
import { createPdf } from './path/to/deno-pdf-generator/mod.ts';
```

## Example Usage

The following example showcases all the major features of the generator.

```typescript
import { createPdf } from './mod.ts';

// Data for our table
const tableBody = [
  [
    { text: 'Feature', alignment: 'center' },
    { text: 'Description', alignment: 'center' },
    { text: 'Example', alignment: 'center' },
  ],
  [
    'Page Setup',
    'Set page size and orientation. This document is A4 Portrait.',
    { text: 'pageSize: "A4", pageOrientation: "portrait"', alignment: 'left' }
  ],
  [
    'Text Wrapping & Alignment',
    'Long lines of text automatically wrap to fit their container. You can align text left, right, or center.',
    { text: 'This text is centered.', alignment: 'center' }
  ],
  [
    'Complex Tables',
    { text: 'Tables support dynamic widths, styling, and advanced cell content.', alignment: 'center', verticalAlignment: 'middle' },
    { text: 'widths: [100, "*", 150]', alignment: 'left' }
  ],
  [
    'Cell Alignment',
    'This cell demonstrates vertical and horizontal alignment within a table.',
    { text: 'Bottom-Right', alignment: 'right', verticalAlignment: 'bottom' }
  ],
  [
    'Multi-Page & Footers',
    'Content automatically flows to new pages. A footer can be replicated on each page with page numbers.',
    { text: 'Page {pageNumber} of {totalPages}', alignment: 'center' }
  ]
];

// Add more rows to test multi-page functionality
for (let i = 1; i <= 15; i++) {
    tableBody.push([`Row ${i}`, `Some content for row ${i}`, 'This is just to make the document longer.']);
}


const docDefinition = {
  pageSize: 'A4',
  pageOrientation: 'portrait',
  footer: {
    text: 'Page {pageNumber} of {totalPages} - Generated by Deno PDF Generator',
    alignment: 'center',
    fontSize: 10,
    color: [0.5, 0.5, 0.5],
  },
  content: [
    {
      text: 'Deno PDF Generator - Feature Showcase',
      fontSize: 24,
      alignment: 'center',
      color: [0.1, 0.6, 0.9],
    },
    {
      text: 'This document demonstrates all the major features of the Deno PDF Generator module. It includes examples of page setup, text formatting, complex tables, and footers.',
      fontSize: 12,
      alignment: 'left',
    },
    {
        image: 'https://placehold.co/500x80.png',
    },
    {
      text: 'Feature Table',
      fontSize: 18,
      alignment: 'center',
    },
    {
      table: {
        widths: [100, '*', 150],
        body: tableBody,
      },
      fontSize: 12,
      layout: {
        fillColor: (rowIndex: number) => (rowIndex === 0) ? [0.8, 0.8, 0.8] : null,
        borderColor: [0.7, 0.7, 0.7],
        borderWidth: 0.5,
      }
    },
  ],
};

async function generate() {
  const pdfBytes = await createPdf(docDefinition);
  await Deno.writeFile('showcase-example.pdf', pdfBytes);
  console.log('Showcase PDF generated successfully!');
}

generate();
```

Run the script using Deno:
```bash
deno run --allow-net --allow-write --allow-import example.ts
```
